Adds additional reporting charts and an activity ticker to the WooCommerce reporting tab.

== Description ==

Get instant insight into trends and activity happening in your store with the additional reporting tab included with this plugin. There are six charts that show activity over the last 30 days as well as an activity ticker that shows the last 30 orders and refunds that have happened in your store.

Charts Include:

*	Daily Revenue Total 
*	Daily Order Count Total
*	Daily Active Customer Count
*	Daily Refunds Total
*	Daily Average Cart Total
*	Daily Coupons Total

The activity ticker gives you a quick list of the last 30 orders and refunds that've happened in your store.
	
The plugin is still new and actively being developed on. Any feedback is very appreciated!

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to WooCommerce reporting tab to view the charts
